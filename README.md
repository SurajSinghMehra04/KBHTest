# Kon Banega Hero

A modern Bollywood-style educational quiz platform with:

- AI-powered quiz generation
- Easy / Moderate / Hard sections
- Points & leaderboard system
- Avatar evolution
- Lottery qualification tracking
- Anti-tab-switch protection
- Login system
- Responsive UI

## Features

### Quiz Engine
- 15 random questions
- 5 Easy (1 point each)
- 5 Moderate (2 points each)
- 5 Hard (3 points each)

### Avatar Evolution
Evolve through:
- 🐣 Novice
- 📚 Learner
- 🔍 Seeker
- ⚔️ Warrior
- 🛡️ Champion
- 🧠 Master
- 👑 Grandmaster
- 🌟 Legend
- 🔱 Mythic
- 🦸 HERO

### Security
- Tab-switch detection
- Window blur detection
- Fullscreen quiz mode
- Anti-cheat warnings

## Deployment

### GitHub Pages
1. Upload all files to a GitHub repository
2. Enable GitHub Pages
3. Set root folder deployment
4. Open your generated GitHub Pages URL
